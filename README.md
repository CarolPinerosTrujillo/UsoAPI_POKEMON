Aplicación desarrollada en JavaScript que consume una API externa (PokéAPI) para obtener y procesar información de Pokémon de forma dinámica. El proyecto implementa funciones asincrónicas con async/await, manejo de datos en formato JSON y lógica para analizar y comparar estadísticas entre múltiples entidades.

Incluye:

Consumo de API REST mediante fetch
Encapsulamiento de lógica en funciones reutilizables
Extracción y manipulación de datos específicos (stats)
Comparación y evaluación basada en criterios definidos
Algoritmos simples para toma de decisiones (búsqueda de valores máximos)

El objetivo es aplicar conceptos fundamentales de desarrollo web como asincronía, estructuras de control, procesamiento de datos y diseño modular, simulando un caso práctico de uso real de APIs en aplicaciones modernas.
